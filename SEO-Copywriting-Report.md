### 2.2 KINDERSCHMINKEN (kinderschminken.html)

#### Hero-Subtitle
| | |
|---|---|
| **IST** | "Märchenhafte Einhörner, funkelnde Feen, furchtlose Superhelden — was darf es heute sein?" |
| **Problem** | Kein Geo-Bezug, kein Vertrauenssignal, kein Preis. Die Frage "was darf es heute sein?" klingt wie ein Eisstand. |
| **SOLL** | "Einhörner, Superhelden, Tiger — jedes Motiv ein Unikat. Professionelles Kinderschminken in Zürich ab CHF 145/h, direkt bei eurem Event." |

#### Feature 1 — Haupttext
| | |
|---|---|
| **IST** | "Jedes Motiv ist ein Unikat — und genau so soll sich jedes Kind fühlen: Einzigartig!" |
| **Problem** | Klischee ("Einzigartig!"), das Ausrufezeichen wirkt aufgesetzt. Der zweite Satz "Ob Prinzessin, Tiger..." ist gut, aber endet unkonkret. |
| **SOLL** | "Jedes Kind bekommt sein persönliches Wunschmotiv — gemalt in Profi-Qualität mit zertifizierten, hautfreundlichen Farben. Vom zarten Schmetterling bis zum feurigen Drachen: Isa bringt die Fantasie jedes Kindes auf die Haut." |

#### Feature 2 — Geburtstage & Partys
| | |
|---|---|
| **IST** | "Egal ob mit oder ohne Thema — dadurch, dass ich alle Motive selbst male, kann ich mich ganz nach euren Wünschen richten." |
| **Problem** | Konstruiert ("dadurch, dass"), wirkt unsicher ("egal ob"). |
| **SOLL** | "Habt ihr ein Partymotto? Perfekt — ich stimme jedes Motiv auf euer Thema ab. Keine Vorlagen, keine Schablonen — alles frei Hand gemalt, individuell für euer Kind." |

#### Feature 3 — Grosse Gruppen
| | |
|---|---|
| **IST** | "Grösseres Event geplant? Sprecht mich einfach an und wir können gemeinsam besprechen, ob es Sinn macht, dass ich Verstärkung zum Kinderschminken mitbringe." |
| **Problem** | Sehr zögerlich formuliert. "ob es Sinn macht" weckt kein Vertrauen. "Verstärkung mitbringe" klingt militärisch. |
| **SOLL** | "Grössere Kindergruppen? Kein Problem. Ab ca. 30 Kindern kann ich mit einer zweiten Fachkraft anreisen — einfach bei der Anfrage angeben, ich organisiere den Rest." |

#### Preis-Section Subtitle
| | |
|---|---|
| **IST** | "Faire Preise für unvergessliche Momente." |
| **Problem** | Nichtssagend. Jeder sagt das. Keine konkreten Zahlen im Teaser. |
| **SOLL** | "Ab CHF 145 / Stunde — alles inklusive, keine versteckten Kosten." |

#### Buchungsablauf Subtitle
| | |
|---|---|
| **IST** | *(kein Subtitle)* |
| **Problem** | Fehlende Orientierung vor dem Schritt-für-Schritt. |
| **SOLL** | "In 3 Schritten zum gebuchten Kinderschminken in Zürich — unkompliziert, schnell, verlässlich." |

### 2.3 HOCHZEITEN (hochzeiten.html)

#### Hero-Subtitle
| | |
|---|---|
| **IST** | "Face Art als romantisches Highlight — Brautpaar und Gäste schwärmen noch lange davon." |
| **Problem** | Passiv. "Schwärmen noch lange davon" ist vage. Was genau wird unvergesslich? |
| **SOLL** | "Elegante Face Art und Body Art für euren Hochzeitstag in Zürich — ein Erlebnis, das Gäste begeistert und auf Fotos unvergesslich aussieht." |

#### Feature 1 — Highlight für Gäste
| | |
|---|---|
| **IST** | "Face Art an einer Hochzeit ist mehr als nur ein Detail — es ist ein Erlebnis. Gäste jeder Altersgruppe stehen Schlange, lachen und staunen gemeinsam." |
| **Problem** | Gut! Aber "stehen Schlange" kann negativ klingen (langes Warten). |
| **SOLL** | "Face Art an einer Hochzeit ist das Gesprächsthema des Abends. Kinder, Eltern, Grosseltern — alle strahlen und staunen gemeinsam um den Schminktisch." |

#### Schritt 2 — Angebot
| | |
|---|---|
| **IST** | "Ich schicke euch ein maßgeschneidertes Angebot, abgestimmt auf euren besonderen Tag." |
| **Problem** | "maßgeschneidert" ist ein abgenutztes Marketing-Wort. Falsche Orthographie: "maßgeschneidert" → Schweiz: "massgeschneidert". |
| **SOLL** | "Ihr erhaltet ein transparentes Angebot mit Preis, Dauer und allem was dazugehört — innerhalb von 24h." |

### 2.4 FÜR UNTERNEHMEN (unternehmen.html)

#### Step 1 — Kontakt
| | |
|---|---|
| **IST** | "Kontaktieren Sie uns mit den Eventdetails: Datum, Thema, Ort und Gästeanzahl." |
| **Problem** | "uns" ist falsch — es ist eine Einzelperson, nicht ein Team. Verwirrend und unpersönlich. |
| **SOLL** | "Schreiben Sie Isa mit den Eckdaten: Datum, Eventthema, Ort und Gästezahl." |

#### Step 3 — Details klären
| | |
|---|---|
| **IST** | "Nach Ihrer Bestätigung klären wir alle Details in einem kurzen (Video-)Call." |
| **Problem** | "wir" ist falsch (Einzelperson). "(Video-)Call" klingt zu Start-up-haft für eine Kunstdienstleistung. |
| **SOLL** | "Nach der Buchung stimmt sich Isa persönlich mit Ihnen ab — per Telefon oder kurzer Nachricht." |

#### Benefit — Brand Integration
| | |
|---|---|
| **IST** | "Lassen Sie Ihr Firmenlogo oder Eventthema kreativ integrieren — einfach aber wirkungsvoll." |
| **Problem** | "einfach aber wirkungsvoll" klingt wie ein Werbespruch aus den 90ern. |
| **SOLL** | "Firmenlogo, Produktfarben oder Eventmotto — Isa integriert Ihre Markenidentität direkt ins Motiv. Jedes Kind trägt Ihr Branding — buchstäblich." |

| Bewertungs-Schema im Hero | Google Sternebewertung | ⭐⭐⭐⭐⭐ (6 Bewertungen) prominent zeigen |

### 3.2 Kinderschminken — Fehlend
| Sektion | Warum wichtig | Empfehlung |
|---|---|---|
| Motiv-Galerie mit Namen | SEO + Conversion | Grid mit Motiv-Namen (Einhorn, Tiger, Schmetterling...) |
| Kinder/h Angabe | Praktische Info = Vertrauen | "Ca. 6–8 Kinder pro Stunde" sichtbar in Preis-Sektion |
| Buchungsbutton nach Preisen | Conversion | CTA direkt nach Preistabelle |

### 3.3 Hochzeiten — Fehlend
| Sektion | Warum wichtig | Empfehlung |
|---|---|---|
| Beispiel-Hochzeit Story | Authentizität | Kurze Fallstudie mit 1–2 Fotos |
| Paketpreise | Einfacheres Budgetieren | "Hochzeit-Paket 2h ab CHF 290" |

### 3.4 Über uns — Fehlend
| Sektion | Warum wichtig | Empfehlung |
|---|---|---|
| Isas persönliche Geschichte | Vertrauen + Differenzierung | Wann gestartet, erste Event, warum Face Art |
| Ausbildung / Zertifikate | Kompetenz-Signal | "Professionell ausgebildet, hautverträgliche Farben zertifiziert" |
